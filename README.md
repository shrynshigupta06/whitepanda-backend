## API Endpoints

> **GET** `/api/v1/cars`

> **POST** `/api/v1/cars`

> **PUT** `/api/v1/cars/:id`

> **DELETE** `/api/v1/cars/:id`

> **GET** `/api/v1/cars/:id`

> **POST** `/api/v1/auth/register`

>  **POST** `/api/v1/auth/login`

>  **POST** `/api/v1/booking`
